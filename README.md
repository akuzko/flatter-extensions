# Flatter::Extensions

A set of extensions to be used with [Flatter](https://github.com/akuzko/flatter) gem.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'flatter-extensions'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install flatter-extensions
